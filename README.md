# PS-19 MediBook

A full-stack healthcare appointment booking platform.

## Features

* Patient Registration & Login
* Doctor Registration & Login
* Admin Panel
* Book Appointments
* View Appointment History
* Search Doctors by Specialization
* PostgreSQL Database
* MongoDB Search Service
* FastAPI Gateway
* Dockerized Deployment

## Tech Stack

* React.js
* Spring Boot
* FastAPI
* Node.js
* PostgreSQL
* MongoDB
* Docker

## Run Project

```bash
docker compose up -d
```

Frontend:

http://localhost:3000

Backend:

http://localhost:8080
